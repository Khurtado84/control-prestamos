control-préstamos.
este proyecto sera una base de datos para el control de prestamos 
a interéses, con pagos mensuales, quincenales, semanales y solo a interés 
tendremos recordatorio de pago vía whatsapp 
se manejara desde la plataforma de Supabase,utilizando SQLpostgress
con el fin de tener una base de datos alojada en la nube, con usuario y contraseña seguro 
también asceso desde cualquier parte
lo que pueden ver aqui es una plantilla funcional para registro, practicas y evaluación
